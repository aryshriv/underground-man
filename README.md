# The Underground Man's Apartment

An interactive 3D experience inspired by Dostoevsky's "Notes from Underground." Walk through the Underground Man's apartment in first-person perspective.

## Features

- **First-person exploration** with WASD movement and mouse look
- **3D audio** including footsteps and atmospheric sounds
- **Realistic physics** with collision detection and jumping
- **Immersive lighting** with candle effects and shadows

## Tech Stack

- **Next.js 16** - React framework
- **Three.js** - 3D graphics
- **TypeScript** - Type safety
- **TailwindCSS** - Styling

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) and click to begin.

**Tip:** Use headphones for the best audio experience.

## Controls

- **WASD** - Move around
- **Mouse** - Look around
- **Space** - Jump
- **Shift** - Walk faster
